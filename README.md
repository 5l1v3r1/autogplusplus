# autogplusplus

Automate g++ in Python3

# Requirements

- Linux

- python3

# Installation

`git clone https://github.com/iinc0gnit0/autogplusplus`

`cd autogplusplus`

`chmod +x insatll.sh`

`./install.sh`

# Usage

`gpp`
